# Billing System Documentation

## Overview

The AquaBill Billing System is a comprehensive water billing management solution built with Laravel 10 and React 18. It provides a complete workflow for managing customer bills, meter readings, and revenue tracking with modern UI components and enhanced user experience.

- **Enhanced Customer Management**: Redesigned customer show page with modern tabs and export functionality
- **Improved User Management**: Complete user edit functionality with modern form design
- **Streamlined Invoice Creation**: Simplified invoice forms with better UX
- **Security Enhancements**: Active user-only login restrictions
- **UI/UX Improvements**: Modern card-based designs and responsive layouts

## Features

### 🏠 Billing Dashboard

- **Real-time Statistics**: View total bills, revenue, pending payments, and overdue bills
- **Revenue Trends**: Monthly revenue charts for the last 12 months
- **Recent Activity**: Latest generated bills and overdue bills requiring attention
- **Top Customers**: Customers with highest water consumption
- **Quick Actions**: Generate bills, process payments, export reports

### 📊 Billing Reports

- **Advanced Filtering**: Filter by date range, status, and area
- **Detailed Analytics**: Total bills, amounts, and averages
- **Export Capabilities**: Export data in CSV and Excel formats
- **Pagination**: Handle large datasets efficiently

### 💰 Bill Generation

- **Automated Calculation**: Generate bills based on meter readings
- **Tariff Management**: Apply different rates and fixed charges
- **Period Management**: Define billing periods with start and end dates
- **Duplicate Prevention**: Avoid generating duplicate bills for the same period

## Architecture

### Backend (Laravel)

#### BillingController

Located at `app/Http/Controllers/BillingController.php`

**Key Methods:**

- `index()` - Dashboard with comprehensive statistics
- `generateBills()` - Generate bills for all customers
- `reports()` - Detailed billing reports with filtering
- `export()` - Export billing data

**Private Helper Methods:**

- `getBillingStats()` - Calculate dashboard statistics
- `getMonthlyRevenue()` - Get 12-month revenue data
- `getTopCustomersByConsumption()` - Top customers by usage
- `getBillingSummary()` - Summary by bill status
- `calculateConsumption()` - Calculate usage from meter readings

#### Models

- **Bill**: Core billing entity with relationships to Customer, Meter, and User
- **Customer**: Customer information with bills and meters
- **Meter**: Meter details with readings and bills
- **Tariff**: Pricing structure for bill calculation

### Frontend (React)

#### Pages

- **Billing Dashboard** (`resources/js/pages/billing/index.jsx`)
    - Comprehensive dashboard with metrics, charts, and recent activity
    - Real-time statistics and quick actions
- **Billing Reports** (`resources/js/pages/billing/reports.jsx`)
    - Detailed reports with filtering capabilities
    - Export functionality and pagination

#### Components

- **BillGenerationModal** (`resources/js/components/billing/BillGenerationModal.jsx`)
    - Modal for generating new bills
    - Date selection and tariff configuration

## Database Schema

### Bills Table

```sql
- id (Primary Key)
- customer_id (Foreign Key)
- meter_id (Foreign Key)
- reading_id (Foreign Key)
- billing_period_start (Date)
- billing_period_end (Date)
- consumption (Decimal)
- unit_price (Decimal)
- fixed_charge (Decimal)
- total_amount (Decimal)
- status (Enum: unpaid, paid, overdue)
- generated_by (Foreign Key to Users)
- created_at, updated_at, deleted_at
```

### Key Relationships

- Bills belong to Customers and Meters
- Bills have one Invoice
- Bills are generated by Users
- Customers have many Meters and Bills
- Meters have many Readings and Bills

## Usage

### Generating Bills

1. **Navigate to Billing Dashboard**

    - Access `/billing` route
    - View current statistics and recent activity

2. **Generate New Bills**

    - Click "Generate Bills" button
    - Select billing period (start and end dates)
    - Choose applicable tariff
    - Confirm generation

3. **Monitor Progress**
    - View generated bills in "Recent Bills" section
    - Check for any overdue bills requiring attention

### Viewing Reports

1. **Access Reports Page**

    - Navigate to `/billing/reports`
    - View summary statistics

2. **Apply Filters**

    - Set date range for billing period
    - Filter by bill status (unpaid, paid, overdue)
    - Filter by customer area
    - Apply filters to see filtered results

3. **Export Data**
    - Use export buttons for CSV or Excel format
    - Download filtered data for external analysis

## API Endpoints

### Billing Routes

```php
GET    /billing                    - Billing dashboard
POST   /billing/generate          - Generate bills
GET    /billing/reports           - Billing reports
POST   /billing/export            - Export billing data
```

### Bill Management Routes

```php
GET    /bills                     - List all bills
POST   /bills                     - Create new bill
GET    /bills/{id}                - Show bill details
PUT    /bills/{id}                - Update bill
DELETE /bills/{id}                - Delete bill
```
